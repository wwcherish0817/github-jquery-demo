• Jq选择器:
  • 基础选择器
  • $("*") 选择所有元素.
  • $(".class") 选择给定样式类名的所有元素
  • $("span")根据给定（html）标记名称选择所有的元素
  • $("#id")选择一个具有给定id属性的单个元素。
  • $("span,p.p1,a,div")将每一个选择器匹配到的元素合并后一起返回
  • 属性选择器
  • 单引号里的双引号：$('a[rel="nofollow self"]')
  • 双引号里的单引号： $("a[rel='nofollow self']")
  • 单引号里面转义单引号： $('a[rel=\'nofollow self\']')
  • 双引号里面转义双引号： $("a[rel=\"nofollow self\"]")
  • 选择指定属性值等于给定字符串或以该字符串为前缀（该字符串后跟一个连字符“-” ）的元素。
    $('a[hreflang|="en"]')
    <a href="example.html" hreflang="en"> jq-select-true </a>
    <a href="example.html" hreflang="en-UK"> jq-select-true </a>
    <a href="example.html" hreflang="english"> jq-select-false </a>
  • 选择指定属性具有包含一个给定的子字符串的元素。（选择给定的属性是以包含某些值的元素）
    $('input[name*="man"]')
    < input name="man-news" /> jq-select-true
    < input name="milkman" /> jq-select-true
    < input name="letterman2" /> jq-select-true
    < input name="newmilk" /> jq-select-false
  • 选择指定属性用空格分隔的值中包含一个给定值的元素。
    $('input[name~="man"]')
    < input name="man-news" /> jq-select-false
    < input name="milk man" /> jq-select-true
    < input name="letterman2" /> jq-select-false
    < input name="newmilk" /> jq-select-false
  • 选择指定属性是以给定值结尾的元素。这个比较是区分大小写的。
    $('input[name$="letter"]')
    < input name="newsletter" /> jq-select-true
    < input name="milkman" /> jq-select-false
    < input name="jobletter" /> jq-select-true
  • 选择指定属性是给定值的元素。
    $('input[value="Hot Fuzz"]')
    < type="radio" name="newsletter" value="Hot Fuzz" /> jq-select-true
    < type="radio" name="newsletter" value="Cold Fusion" /> jq-select-false
    < type="radio" name="newsletter" value="Evil Plans" /> jq-select-false
  • 选择不存在指定属性，或者指定的属性值不等于给定值的元素。
    $('input[name!="newsletter"]')
    < type="radio" name="newsletter" value="Hot Fuzz" /> jq-select-false
    < type="radio" name="newsletter" value="Cold Fusion" /> jq-select-true
    < type="radio" name="newsletter" value="Evil Plans" /> jq-select-true
  • 选择指定属性是以给定字符串开始的元素
    $('input[name^="news"]')
    < input name="newsletter" /> jq-select-true
    < input name="milkman" /> jq-select-false
    < input name="newsboy" /> jq-select-true
  • 选择所有具有指定属性的元素，该属性可以是任何值。
    $( "div[id]" )
    <div> no id </div> jq-select-false
    <div id="hey" > id ="hey" </div> jq-select-true
    <div id="there"> id ="there" </div> jq-select-true
    <div> no id </div> jq-select-false
  • 选择匹配所有指定的属性筛选器的元素
    $('input[id][name$="man"]')
    < input id="man-news" name="man-news" /> jq-select-false
    < input name="milkman" /> jq-select-false
    < input id="letterman" name="new-letterman" /> jq-select-true
    < input name="newmilk" /> jq-select-false
  • 基础筛选选择器
  • 选择所有正在执行动画效果的元素.
    $("div:animated")
  • 在匹配的集合选择索引值为n的元素。 从0开始
    $("td:eq(2)")
  • 选择偶数元素，从 0 开始计数
    $("tr:even")
  • 选择第一个匹配的元素。
    $("tr:first")
  • 选择当前获取焦点的元素。
    <div id="content">
      <input tabIndex="1">
      <input tabIndex="2">
      <select tabIndex="3">
       <option>select menu</option>
      </select>
     <div tabIndex="4">
        a div
      </div>
     </div>
    <script>
    $( "#content" ).delegate( "*", "focus blur", function( event ) {
    var elem = $( this );
    setTimeout(function() {
    elem.toggleClass( "focused", elem.is( ":focus" ) );
    }, 0);
    });
    </script>
  • 选择匹配集合中所有大于给定index（索引值）的元素。
    $( "td:gt(4)" ) 从0开始计数的索引值。
    $( "td:gt(-2)" ) 从0开始计数的索引值。 从最后一个元素开始反向计数。
  • 选择所有标题元素，像h1, h2, h3 等.
    $(":header")
  • 选择指定语言的所有元素。
    $( "div:lang(en-us)" )
    <div lang="en-us">red </div>
  • 选择最后一个匹配的元素。
    $("tr:last")
  • 选择匹配集合中所有小于给定index索引值）的元素。
    $("td:lt(4)") 从 0 开始计数的索引值。
    $("td:lt(-2)") 从0开始计数的索引值。 从最后一个元素开始反向计数。
  • 选择所有元素去除不匹配给定的选择器的元素。
    $("input:not(:checked) + span").css("background-color", "yellow")
    <div> jq-select-true
     <input type="checkbox" name="a" />
     <span>Mary</span> color-yellow
    </div>
    <div> jq-select-true
     <input type="checkbox" name="b" />
     <span>lcm</span> color-yellow
    </div>
    <div> jq-select-false
     <input type="checkbox" name="c" checked="checked" />
     <span>Peter</span>
    </div>
  • 选择奇数元素，从 0 开始计数。
    $("tr:odd")
  • jQuery( ":root" ) 选择该文档的根元素。 $( ":root" )[0].nodeName == "HTML"
  • 内容筛选选择器
  • 选择所有父级元素下的第一个子元素。
    $("div span:first-child")
  • 选择所有相同的元素名称的第一个兄弟元素。
    $("span:first-of-type")
  • 选择所有父级元素下的最后一个子元素
    $("div span:last-child")
  • 选择所有相同的元素名称的最后一个兄弟元素。
    $("span:last-of-type")
  • 每个相匹配子元素的索引值，从1开始，也可以是字符串 even 或 odd，或一个方程式( 例如 :nth-child(even), :nth-child(4n) )。
    $("p:nth-child(2)")
  • 每个相匹配子元素的索引值，从最后一个开始(1)，也可以是字符串 even 或 odd，或一个方程式( 例如 :nth-last-child(even), :nth-last-child(4n) )
    $("ul li:nth-last-child(2)")
  • 每个相匹配子元素的索引值，从最后一个开始(1)，也可以是字符串 even 或 odd，或一个方程式( 例如 :nth-last-of-type(even), :nth-last-of-type(4n))
    $("ul li:nth-last-of-type(2)")
  • 每个相匹配子元素的索引值，从1开始，也可以是字符串 even 或 odd，或一个方程式( 例如 :nth-of-type(even), :nth-of-type(4n))。
    $( "span:nth-of-type(2)" )
  • 选择所有其父元素下只有一个子元素的元素。
    $("div button:only-child")
  • 选择所有元素没有兄弟元素具有相同的元素名称的所有元素。
    $("button:only-of-type")
  • 内容过滤选择器
  • 用来查找的一个文本字符串。这是区分大小写的。
    $("div:contains('John')")
  • 选择所有没有子元素的元素（包括文本节点）
    $("td:empty")
  • 选择元素其中至少包含指定选择器匹配的一个种元素。任何选择器。
    $("div:has(p)")
  • 选择所有含有子元素或者文本的父级元素
    $("td:parent")
  • 选择所有按钮元素和类型为按钮的元素。
    $(":button")
  • 选择所有类型为复选框的元素。
    $("form input:checkbox")
  • 匹配所有勾选的元素。
    $( "input:checked" )
  • 选择所有被禁用的元素。
    $("input:disabled")
  • 选择所有可用的元素。
    $("input:enabled")
  • 选择所有类型为文件（file）的元素。
    $("input:file")
  • 选择所有图像类型的元素。
    $("input:image")
    <input type="image" />
  • 选择所有 input, textarea, select 和 button 元素.
    $(":input")
  • 选择所有类型为密码的元素。
    $("input:password")
  • 选择所有类型为单选框的元素。
    $("input:radio")
  • 选择所有类型为重置的元素。
    $("input:reset")
    <input type="reset" />
  • 选择所有被选中的元素。
    $("select option:selected")
  • 选择所有类型为提交的元素。
    $("input:submit ")
    <input type="submit" />
  • 选择所有类型为提交的元素。
    $("input:text ")
    <input type="text" />
  • 层级选择器
  • 选择所有指定“parent”元素中指定的"child"的直接子元素。
    $("ul.topnav > li")
  • 选择给定的祖先元素的所有后代元素。
    $("form input")
  • 选择所有紧接在 “prev” 元素后的 “next” 元素
    $("label + input")
    <label>Name:</label>
    <input name="name" />
  • 匹配 “prev” 元素之后的所有 兄弟元素。具有相同的父元素，并匹配过滤“兄弟姐妹”选择器。
    $("#prev ~ div")
